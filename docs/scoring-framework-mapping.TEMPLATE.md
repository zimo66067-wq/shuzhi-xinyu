## 填写须知

1. "拟对应框架"一格若经查证**无合适映射**，请如实写"无对应循证条目"，**不要硬凑**。
2. "文献 / 量表出处"必须是可核实的真实来源（量表名 + 版本 + 条目编号 / 文献引用），禁止虚构编号。
3. "信效度说明"需说明该映射是否有信度 / 效度证据；若仅为团队启发式推断，须明确标注"无信效度证据，仅为启发式"。
4. 全表填写并经至少一位具相关知识者复核前，本文件状态为"未完成"，产品对外定位维持"参考性启发式、非诊断、非分级"。
5. 完成后可将本文件重命名去掉 `.TEMPLATE`，并在 `docs/competitive-gap-report.md` 2.1.1 处回链。
