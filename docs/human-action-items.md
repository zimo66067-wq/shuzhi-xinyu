# 人工 / 外部待办清单

> 来源：docs/competitive-gap-report.md
> 这些事项**不是写代码能解决的**——需要法务、临床、商业、部署或真机测试等人工/外部资源。
> 每条标注：负责类型 + 需要什么。

---

## C-1 香港免备案 HTTPS 演示链路（比赛刚需）
- **对应报告**：2.6.1
- **负责类型**：部署 / 运维
- **需要什么**：买一台阿里云香港地域轻量服务器（免 ICP 备案），按 `docs/DEPLOYMENT.md` 部署，配 HTTPS（Let's Encrypt）。目的是现场能连一个真实 HTTPS 地址（HTTP 下浏览器禁用麦克风）。同时准备本机 localhost 演示作为备份。

## C-2 iOS Safari / 桌面真机过一遍语音链路
- **对应报告**：2.3.1
- **负责类型**：测试（真机）
- **需要什么**：用 iPhone Safari + 安卓 Chrome + 桌面 Chrome/Edge 各跑一遍"长按说话 → 识别 → 回复 → TTS"。重点确认 iOS 产出的音频编码（webm 还是 mp4）能被后端 ffmpeg + 腾讯云 ASR 正确处理（与 B-3 联动）。记录每端结果。

## C-3 与 DeepSeek / 腾讯云谈"不用于训练" / 签 DPA
- **对应报告**：2.2.2
- **负责类型**：法务 / 商务
- **需要什么**：核对 DeepSeek、腾讯云最新企业条款是否提供"数据不用于模型训练"选项；商业部署前签订数据处理协议（DPA）。结论回填 `PRIVACY.md` 第 3 节第三方数据流向。

## C-4 机构试点 + 临床联合署名
- **对应报告**：2.6.2 / 2.6.3
- **负责类型**：商业 / 临床合作
- **需要什么**：联系 1-2 家康复中心 / 特教学校做免费试点，拿真实使用案例；争取一位持证治疗师 / 特教专家联合署名或共研，获取临床背书（这是大厂最难快速复制的稀缺资产，也是机构采购的信任前提）。理清公立机构政府采购 / 企业服务年费流程。

## C-5 大陆真机 DevTools Network 截图存档
- **对应报告**：2.3.3
- **负责类型**：测试（真机）
- **需要什么**：在大陆网络真机上跑完整流程，开 DevTools Network 面板确认**无任何对 gstatic / googleapis 等外网的请求、无 timeout 失败请求**，截图存进 `docs/`，作为"无墙依赖、大陆可用"的硬证据。（代码层已本地托管 MediaPipe、关闭 draco/meshopt、前端零外网 CDN，此项是补证据。）

## C-6 PRIVACY.md / 知情同意文案法务复核
- **对应报告**：2.2.1 / 2.2.2
- **负责类型**：法务
- **需要什么**：本轮已实现"监护人知情同意页"（`ConsentGate.jsx`）与 PRIVACY 必要披露，但**全文合规性需法务复核**：是否满足《个人信息保护法》对不满 14 周岁未成年人的"单独同意 + 专门处理规则"、留存周期、删除权、跨境传输等要求。法务确认前，对外不得宣称"已合规"。

## C-7 危机求助热线号码核实填入
- **对应报告**：2.1.3
- **负责类型**：内容 / 合规
- **需要什么**：`ParentPage` 危机求助区块、`ConsentGate` 与文档中的热线号码当前为占位符「【填入已核实的本地心理援助热线，勿编造】」。需由人工查证**真实、当前有效**的本地心理援助热线 / 急救求助电话后填入。**严禁编造号码。**

## C-8 报告第 5 节"不确定项"逐项调研
- **对应报告**：竞品报告第 5 节
- **负责类型**：研发 / 测试 / 法务（按项）
- **需要什么**：逐项落实——
  1. 后端是否真零落盘 / 零日志（审 `app.py` / `stt_service.py` / `tts_service.py` 实际写盘与日志；生产 Nginx 访问日志是否含敏感参数）。
  2. DeepSeek / 腾讯云"用于训练"与留存条款核对（与 C-3 合并）。
  3. iOS 语音链路真机验证（与 C-2 合并）。
  4. 评分信度量化（同段对话多次评分方差，与 B-1 联动）。
  5. 18MB VRM 模型在评委网络/设备的加载耗时与 WebGL 兼容性实测。
  6. 限流 / CORS / CSP 真实配置值是否合理、是否覆盖所有敏感端点。
  7. 精确竞品对标需另行专项调研（仅用公开可核实信息，不得编造）。
