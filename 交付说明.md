## 这是什么

「数智心屿」是一个面向自闭症谱系儿童的 **AI 数字人陪伴式社交对话练习工具**。
数字人「心屿」用语音 + 文字 + 表情和孩子自然对话，并按训练阶段引导。

> ⚠️ 本工具仅供研究 / 教育 / 家庭辅助使用，**不是医疗器械，不能用于诊断或治疗**。详见 [README.md](./README.md) 顶部声明。

## 怎么跑起来（本地）

### 环境要求
- Node.js 18+（推荐 20+）
- Python 3.10+
- FFmpeg（已加入系统 PATH）

### 第 1 步：启动后端

```bash
cd backend
python -m venv venv
venv\Scripts\activate          # Windows
pip install -r requirements.txt
python app.py                  # 监听 http://localhost:5000
```

> Windows 若 `python` 无效，改用 `py -X utf8 app.py`

### 第 2 步：启动前端（新开一个终端）

```bash
cd frontend
npm install                    # 首次必须，下载依赖（约 2-5 分钟）
npm run dev                    # 监听 http://localhost:5173
```

### 第 3 步：打开浏览器

访问 **http://localhost:5173**

体验流程：输入孩子姓名+年龄 → 和心屿对话（可语音可打字）→ 工具盒里有「超市购物练习」场景 → 长按管理员入口输家长密码看评分报告。

## ⚠️ 安全提醒（重要）

1. **本包内 `backend/.env` 含真实 API 密钥**（DeepSeek + 腾讯云）。
   - 请勿将本包公开上传到 GitHub / 网盘公开链接
   - 演示结束后，原作者会轮换这批密钥
2. **家长后台默认密码**见 `backend/.env` 里的 `PARENT_PASSWORD`。
3. 生产部署请务必走 HTTPS（否则浏览器禁用麦克风）—— 见 [docs/DEPLOYMENT.md](./docs/DEPLOYMENT.md)。
