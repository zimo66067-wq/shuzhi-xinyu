# 数据与隐私说明（PRIVACY）

> 本文档说明「数智心屿」处理用户数据的方式。
> 如果你打算把本项目部署给真实用户使用，请仔细阅读、并根据你所在地区的法律（GDPR / China PIPL / COPPA 等）调整后再用。

---

## 一句话总结

**本项目不主动收集任何个人数据。** 孩子的姓名、年龄、对话历史均仅保存在**本地浏览器** (localStorage)；对话文本和语音会通过你部署的后端实时转发给 DeepSeek / 腾讯云 API 完成 AI 推理与语音处理，**后端默认不留存任何内容**。

| 角色 | 说明 |
|---|---|
| 项目作者 | 仅提供开源代码，**不运营任何在线服务**，不接触任何用户数据 |

## 2. 数据分类及处理

### 2.1 仅本地存储（localStorage，不离开用户设备）

| 数据 | 键名 | 用途 |
|---|---|---|
| 孩子姓名、年龄 | `xinyu_childInfo` | 心屿对话称呼 + 年龄分档 |
| 对话历史 | `xinyu_messages` | 跨页面恢复对话 |
| 声学摘要（最近 50 条） | `xinyu_acousticHistory` | 家长后台展示 |
| 当前页面 | `xinyu_currentPage` | 刷新后恢复 |

**清除方式**：浏览器开发者工具 → Application → Local Storage → 删除对应键；或对话页"清空所有记录"按钮。

### 2.2 实时转发，不留存

| 数据 | 流向 | 说明 |
|---|---|---|
| 对话文本 | 前端 → 部署者后端 → **DeepSeek 服务器** | 用于生成回复 / 评分 / 报告 |
| 语音录音 (webm/opus) | 前端 → 部署者后端 → **腾讯云 ASR** | 转写为文字后丢弃音频 |
| TTS 文本 | 前端 → 部署者后端 → **腾讯云 TTS** | 合成音频后返回前端 |

**默认行为**：后端 `backend/app.py` 的 `/api/stt` 和 `/api/tts` 接口处理完即返回，**不写磁盘、不写日志**。

> ⚠️ 如果部署者修改了后端添加日志或数据库，请在自己的隐私政策中明确披露。

### 2.3 不收集

- ❌ 不收集 IP 地址（除 Nginx 访问日志，部署者自行决定是否保留）
- ❌ 不收集设备指纹
- ❌ 不放第三方分析 SDK（Google Analytics、百度统计等）
- ❌ 不放第三方广告
- ❌ 不向用户索取联系方式

| 第三方 | 数据 | 服务地区 | 隐私政策 |
|---|---|---|---|
| **DeepSeek** | 对话文本（含孩子姓名、年龄、对话内容） | 中国大陆 | https://www.deepseek.com/privacy |

- 商业部署前，**建议与上述厂商签订正式的数据处理协议（DPA）**

## 4. 儿童保护

本项目主要受众是 3-10+ 岁儿童。COPPA / GDPR-K / 中国《未成年人网络保护条例》均对处理儿童数据有更严格的要求。

部署者必须：

1. ✅ **获得监护人明确知情同意**（建议在启动页面加上同意勾选项）
2. ✅ **不要求孩子提供姓名以外的个人信息**（已遵守：本项目只采集名字 + 年龄）
3. ✅ **不向儿童投放广告**（已遵守）
4. ✅ **不分析孩子内容用于商业画像**（已遵守）
5. ✅ **提供随时删除数据的方式**（已遵守：清空 localStorage）

## 5. 安全实践

### 5.1 API 密钥

- 后端 `.env` 文件**绝不提交到版本控制**（已在 `.gitignore`）
- 前端代码不包含任何 API key（Flask 后端作为安全代理）
- 部署到云时，建议把密钥放进服务商提供的密钥管理服务

### 5.2 家长密码

- 默认密码 `1234` 仅供首次试用
- 通过 `VITE_PARENT_PASSWORD` 环境变量自定义
- **本项目的家长密码是简单的客户端校验，不是认证系统**。任何能查看页面源码的人都能看到密码值
- 如果场景需要严肃的权限隔离，请基于后端实现真正的会话认证

### 5.3 传输加密

- 开发环境 (HTTP localhost) 没有 TLS，仅限本机使用
- **生产部署必须使用 HTTPS**（浏览器麦克风 API 也要求 HTTPS）

## 6. 数据删除

| 数据位置 | 删除方法 |
|---|---|
| 浏览器 localStorage | 对话页"清空所有记录"按钮；或 DevTools 手动清 |
| DeepSeek 服务器 | 按 DeepSeek 隐私政策提交删除请求 |
| 腾讯云 ASR/TTS | 按腾讯云隐私政策提交删除请求；腾讯云一般在 7-30 天内自动清理临时数据 |
